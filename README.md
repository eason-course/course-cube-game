# Cube Game

2026/02/04 嘉義高商 - Unity 工作坊

## 課程簡介

這是一個 Unity 教學專案，我們會用一款已經做好基礎功能的 3D 方塊平台遊戲，帶你學會如何設計與建立自己的遊戲關卡。

## 玩法與操作

| 按鍵 | 動作 |
|------|------|
| W / A / S / D | 移動方塊 |
| 空白鍵 | 跳躍 |
| R | 重新開始關卡 |

遊戲目標：收集關卡中的金幣，閃避陷阱，抵達終點進入下一關。

## 學習重點

### 1. 擺放關卡物件

將現成的 Prefab（玩家、金幣、陷阱、終點）拖進場景中，學習如何調整位置與數量來組成一個完整的關卡。

### 2. 搭建關卡環境

使用 Unity 內建的 3D 物件（Cube 等）自己建立牆壁、高台等地形，設計出屬於自己的關卡路線與挑戰。

### 3. 建立禁止區域

在場景中的任意物件上掛載陷阱腳本（Trap），製作玩家不能碰觸的危險區域，例如毒地板、危險牆面等，為關卡增加更多變化。

### 4. 多關卡切換

建立全新的場景作為新關卡，自由運用前面學到的所有物件與技巧，並將場景加入 Build Settings 讓玩家可以依序闖關。

## 開發環境

- Unity 6（URP）
- Input System
- TextMeshPro
